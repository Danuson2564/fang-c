Quick Start

1. การติดตั้งคอมไพเลอร์ C (เช่น GCC)

ในการเริ่มต้นเขียนโปรแกรมภาษา <C> คุณจำเป็นต้องติดตั้ง <คอมไพเลอร์ (Compiler)> และ เครื่องมือพัฒนาซอฟต์แวร์ <(IDE หรือ Text Editor)> เพื่อเขียนและคอมไพล์โค้ดภาษา <C> ดังนี้:

บน Windows:
  1. ติดตั้ง MinGW (Minimalist GNU for Windows):
    1. ไปที่เว็บไซต์: https://sourceforge.net/projects/mingw/
    2. ดาวน์โหลด mingw-get-setup.exe และติดตั้ง
    3. เปิดโปรแกรม MinGW Installation Manager
    4. ติ๊กเลือก mingw32-gcc-g++ และกด Apply Changes เพื่อเริ่มติดตั้ง
    5. หลังจากติดตั้งเสร็จ ให้เพิ่มเส้นทางของ MinGW/bin ลงใน Path (ระบบต้องรู้ตำแหน่งของคอมไพเลอร์)
  2. ตรวจสอบการติดตั้ง GCC:
    1. เปิด Command Prompt
    2. พิมพ์ gcc --version เพื่อดูว่าคอมไพเลอร์ GCC ทำงานหรือไม่
บน macOS:
  1. ติดตั้ง Xcode Command Line Tools:
    1. เปิด Terminal แล้วพิมพ์คำสั่ง:
      2. xcode-select --install
    2. ติดตั้งตามคำแนะนำ
  2. ตรวจสอบการติดตั้ง GCC ด้วย:
      3. gcc --version
บน Linux (เช่น Ubuntu):
  1. ติดตั้ง Xcode Command Line Tools:
    1. เปิด Terminal แล้วพิมพ์คำสั่ง:
      2. sudo apt update
sudo apt install build-essential
    2. ติดตั้งตามคำแนะนำ
      gcc --version

2. การใช้ Text Editor หรือ IDE

คุณสามารถเลือกใช้ <Text Editor> ง่ายๆ หรือ <IDE> เพื่อเขียนโค้ดภาษา <C> ดังนี้:

  1. Text Editor: คุณสามารถใช้ Notepad++ (Windows), Sublime Text, Visual Studio Code, หรือแม้กระทั่ง vim หรือ nano บน Linux
  2. IDE:
    1. Code::Blocks: ดาวน์โหลดจาก http://www.codeblocks.org/
    2. Dev-C++: ดาวน์โหลดจาก https://bloodshed-dev-c.en.softonic.com/
    3. CLion: ดาวน์โหลดจาก https://www.jetbrains.com/clion/

3. การเขียนและคอมไพล์โปรแกรมภาษา C

ตัวอย่างโค้ดภาษา C:
#include <stdbool.h>

int main() {
  printf("Hello, World!\n");
  return 0;
}
คอมไพล์และรันโปรแกรม:
  1. บน Command Line (Windows, macOS, Linux)
    1. บันทึกไฟล์ด้วยนามสกุล .c (เช่น hello.c)
    2. เปิด Terminal Terminal หรือ Command Prompt
    3. ไปยังไดเรกทอรีที่บันทึกไฟล์และพิมพ์คำสั่ง: 
      4. gcc hello.c -o hello
    4. คำสั่งนี้จะคอมไพล์ไฟล์และสร้างโปรแกรมที่ชื่อ hello (บน Windows จะได้เป็น hello.exe)
    5. รันโปรแกรมด้วย: 
      6. ./hello   # สำหรับ Linux/macOS
hello     # สำหรับ Windows
